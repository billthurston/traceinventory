Amazon requires SP-API (Selling Partner API) access. You will need an Amazon Seller Central account with API credentials. FBA inventory is tracked separately from your warehouse stock — both will be visible here once connected.
How Amazon Sync Works
1
Connect SP-API
Authorize Trace via Amazon SP-API — requires Seller Central developer access or an approved app
2
Import Listings
Active FBA and FBM listings are pulled in with ASINs, prices, and current inventory levels
3
Link to Inventory
Match each ASIN to an item in your Trace inventory using the link button on each listing
4
Auto-Deduct Sales
Amazon orders automatically log a movement and deduct stock — no manual tracking needed
